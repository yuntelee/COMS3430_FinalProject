Nature‑Inspired Algorithmic Composer

This page pre‑analyzes a selected nature recording (rain / ocean / forest / birds) to extract dominant frequencies, event timings, and texture features. Those sequences train three Markov chains (pitch, timing, texture). You can then morph from ambience → melody using the entropy slider.

To run: save this file as nature-composer.html and put audio files in ./sounds/: rain.mp3, ocean.mp3, forest.mp3, birds.mp3. Serve via a local server (recommended): python -m http.server.
